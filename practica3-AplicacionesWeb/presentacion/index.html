HTML5

HTML5 es el estándar y la piedra angular de la plataforma web abierta de la W3C, diseñado como un framework para apoyar la innovación y fomentar el potencial que ofrece la web. Con un conjunto de herramientas y normas, HTML5 provee elementos visuales para clasificar y comunicar claramente nuestros elementos y contenido...

Semántica

Se da sentido a la estructura, haciendo de la semántica la parte frontal y central de HTML5. Contiene un conjunto más amplio de variables que, en conjunto con RDFa, microdatos, microformatos permiten una mayor utilidad de los datos por parte de los programas y de los usuarios.

Offline y Almacenamiento

Nuestras apps serán más rápidas y e incluso pueden trabajar si no hay conexión a Internet, gracias al HTML5 App Cache, así como el almacenamiento local, DB indexado, y las especificaciones de la API de archivos.

Acceso a Dispositivos

HTML5 cuenta con una API de geolocalización además de innovaciones de acceso a los dispositivos los cuales se están desarrollando y aplicando, desde el acceso de audio / vídeo, acceso a micrófonos y cámaras, a los datos locales, como contactos y eventos, así como a la orientación del dispositivo.

Conectividad

Conectividad más eficiente significa conversaciones en tiempo real, juegos más rápidos, y una mejor comunicación. Sockets Web y envio de eventos al servidor transmiten datos entre el cliente y el servidor de manera más eficiente que nunca.

Multimedia

Audio y video son componentes principales dentro de HTML5, que viven en armonía con las aplicaciones y sitios.

3D, Gráficos y Efectos

SVG, Canvas, WebGL, CSS3 3D ... sorprenderán a los usuarios con impresionantes efectos visuales de forma nativa presentados en el navegador.

Rendimiento e Integración

Aplicaciones Web y contenido Web dinámico más rápido con una variedad de técnicas y tecnologías como Web Workers y XMLHttpRequest 2. La idea: ya no más tiempos de espera.

CSS3

CSS3 ofrece una amplia gama de estilos y efectos, mejorando las aplicaciones web sin sacrificar su estructura o funcionamiento semántico. Además Web Open Font Format (WOFF) proporciona flexibilidad y control tipográfico más allá de lo que la web ha ofrecido antes.
