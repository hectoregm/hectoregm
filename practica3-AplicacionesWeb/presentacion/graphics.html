Canvas

API Canvas

Siendo uno de los componenetes más novedosos de la nueva especificación, la API CANVAS ofrece poderosas características a HTML5 y es quizá, una de las APIs más extensas que la integran.

La API Canvas nos permite dibujar y crear/presentar gráficos que podemos animar y procesar junto con texto. Los elementos generados o manipulados mediante esta API pueden trabajar/interactuar (son compatibles con las implementaciones) con el resto de las APIs que integran la nueva especificación

Generando el Lienzo: <canvas>

Mediante el tag <canvas> generamos el lienzo de trabajo: un espacio rectangular con fondo blanco donde se mostrarán los elementos. Podemos definir algunos atributos:

  • id: Identidicador (único dentro de nuestro documento/página) de nuestro elemnto
  • width: valor entero que define el ancho (en pixeles) de nuestro espacio de trabajo
  • height: valor entero que define la altura (en pixeles) de nuestro espacio de trabajo

Es importante señalar que los elementos serán generados mediante nuevos métodos Javascript incorporados por la especificación, por lo cual es importante definir el atributo id para acceder al elemento.

Por otro lado podemos definir texto (u otros elementos) dentro del tag <canvas> : el contenido dentro del tag será mostardo en caso de que el navegador no sea compatible con el tag 
<canvas id="firstCanvas" width="500" height="300">
   <p>
      El navegador no es compatible con la especificación de canvas
   </p>
</canvas>

Generando el contexto: getContext()

Necesitamos crear un script ( Javascript ) en donde lo primero que se necesita hacer es definir el contexto de dibujo en el cual se trabajará.

Existen dos ambientes de trabajo (contextos de dibujo): '2d' o el ambiente de dos dimensiones y '3d' o el ambiente de tres dimensiones.

Para especificar el contexto de dibujo hacemos uso del método getContext() haciendo referencia al elemento sobre el cual se va a trabajar (por eso la importancia de especificar el valor del atributo id) y pasando como parámetro el contexto de dibujo ('2d' o '3d') 

	
//El siguiente código forma parte del cuerpo de un script								
								
  //Esta función la llamaremos cuando se cargue la pagina
function init(){
	
    //Obtenemos la referencia al canvas mediante su id
  var refCanv=document.getElementById('firstCanvas');
  
    //especificamos el contexto de dibujo (en este caso 2d)
  firstCanvas=refCanv.getContext('2d');
}

 //Mediante este metodo llamamos a la funcion init al cargar la pagina
window.addEventListener("load", init, false);

En nuestro caso, el contexto de dibujo será un area de 500 de ancho por 300 de alto con origen en la esquina superior izquierda [que será la coordenada (0,0)] en donde los elementos serán definidos de arriba hacia abajo y de izquierda a derecha. El resultado del código anterior es el siguiente:

Es importante mencionar que los elementos se dibujar´n uno sobre otro de acuerdo al orden definido en el código de la implementación
