Multimedia

En la nueva especificación se tiene soporte para audio y video y en est sección mostraremos un uso básico de estos elementos

Para agregar audio hacemos uso del tag <audio> y especificamos dentro la ubicación de nuestro archivo con el tag <source>
Podemos definir ademas el ancho que queremos que tenga nuestro reproductor, si queremos que automáticamente inicie la reproducción o si queremos que se muestren los controles

Para agregar video hacemos uso del tag <video> y especificamos dentro la ubicación de nuestro archivo con el tag <source>
Podemos definir ademas el ancho y alto que queremos que tenga nuestro reproductor, si queremos que automáticamente inicie la reproducción o si queremos que se muestren los controles

Todos estos elementos se pueden personalizar o modificar mediante Javascript
Tenga en cuenta que no todos los formatos son soportados por los navegadores y que algunos de ellos (navegadores) no soportan estos tags
